# Camore-suites-airbnb-market-place
amore Suites is a simple Airbnb-style website that showcases available rental units in locations such as Kilimani, GTC, Tatu City, and Tsavo. The project helps users quickly view different accommodation options and navigate to individual unit pages for more details.

## Technologies Used

- HTML for the website structure and page content
- CSS for styling, layout, colors, and visual presentation
- Font Awesome for icons used in the navigation/header

## Problem It Solves

Many travelers want a fast and simple way to compare short-stay rental options before making a booking decision. This website solves that problem by presenting multiple units in one place, with clear navigation to each property page. It makes it easier for visitors to discover available homes without searching through scattered information.

## Market Gap

The project bridges the gap between property owners and guests who need reliable, stylish, and convenient short-term accommodation. Instead of relying only on large booking platforms, Camore Suites creates a direct online presence where units can be promoted clearly, helping guests find suitable stays and helping hosts market their spaces more effectively.
